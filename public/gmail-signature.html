✨ SpontiCoupon Gmail Signature

  1. Open Gmail → Settings (gear) → See all settings → Signature → Create new.
  2. Name it SpontiCoupon.
  3. Click inside the card below, select everything (Cmd A won't work — click & drag), and copy.
  4. Paste into the Gmail editor. Set as default for New emails and Replies.
  5. Click Save Changes.
